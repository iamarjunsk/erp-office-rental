## 2024-05-24 - Replace Native Alerts with shadcn Toasts | Learning: Native `alert()` calls block user interaction and provide a jarring UX, contradicting modern web design standards. The shadcn-vue `useToast` composable offers a non-blocking, visually consistent alternative but does not provide an `.error()` helper, requiring the `variant: 'destructive'` parameter. | Action: Replaced `alert()` with `toast({ title: 'Error', description: ..., variant: 'destructive' })` in `create.vue` to improve the error feedback micro-interaction.
