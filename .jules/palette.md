## 2024-03-29 - Replace Native Alerts with Toasts | Learning: Native `alert()` calls are disruptive and break the modern application feel, particularly when displaying structured API validation errors (e.g., `title: This field is required., description: Description is too short.`). | Action: Always use the `useToast` composable from `shadcn-vue` for error handling. When formatting multi-line validation errors from an API, join them with commas (`, `) instead of newlines (`\n`) so they render correctly within the toast description UI.
