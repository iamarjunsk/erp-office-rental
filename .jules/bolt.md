## 2026-04-19 - Optimize stats query aggregation\n**Learning:** When using Django ORM with aggregate and Count, you can filter counts directly within the database instead of creating multiple hits using Q objects and filter parameters. The syntax `Count('pk', filter=Q(status='open'))` significantly improves performance.\n**Action:** Always favor a single aggregate query with conditional Count filters over multiple count queries when aggregating values from the same model.
